IDE Compiler 

An interactive browser-based IDE inspired by Visual Studio Code, featuring a modern code editor, real-time code execution, execution visualization, and an AI-powered programming tutor. The project combines a React + Monaco Editor frontend with a FastAPI backend that securely executes code inside Docker containers.
